# arduino_pwm_uart
generating PWM signal via UART
for controlling DC motor RPM


# Requaerment :
  1. one arduino borad as master
  2. one arduino board as slave
  3. 10 k potentiameter
  4. L298 motor driver
  5. 12V motor
